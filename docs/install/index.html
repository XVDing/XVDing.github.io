    Install QMC #

    Simple required: Quantum espresso, EPW, wannier90, python3 #

    1. install Quantum espresso
    insure you have required compliers for Quanutm espresso 
./configure 
make pw pp ph
    1. install wannier90
    make w90
    1. install EPW
    make epw
    1. install python3 (need numpy, matplotlib)
    install Anaconda3 or miniconda3
pip install numpy matplotlib
